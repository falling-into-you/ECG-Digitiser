# 数据集构建指南：适配真实世界 ECG 图像

## 问题分析

基于 121 张真实世界 ECG 图像和 400 张训练数据的分析：

### 像素分布对比

| 数据集 | Mean 均值 | Mean 范围 | 分布区间 |
|--------|----------|-----------|---------|
| **Clean (训练)** | 228.7 | [221, 232] | 100% 在 200-255 |
| **Aug (当前增强)** | 122.1 | [82, 169] | 78% 在 100-150 |
| **Real World** | 177.6 | [124, 244] | 67% 在 150-200 |

### 关键问题

1. **Clean 全是白底** (mean=228)，无任何变化
2. **Aug 增强后变暗了** (mean=122)，与真实数据相反
3. **真实数据主要在 150-200 区间**，但训练数据在这个区间几乎没有样本

```
Mean 分布:
区间        Clean    Aug      Real World
─────────────────────────────────────────
0-50         0%       0%       0%
50-100       0%      14.5%     0%
100-150      0%      78.0%    14.0%   ← Aug 集中这里
150-200      0%       7.5%    66.9%   ← Real 集中这里 (GAP!)
200-255    100%       0%      19.0%   ← Clean 全在这里
```

## 解决方案

### 新的数据集构成

| 子集 | 数量 | 目标 Mean | 覆盖区间 | 增强策略 |
|------|------|----------|---------|---------|
| Clean | 5,000 | ~228 | 200-255 | 无增强 |
| Aug Light | 8,000 | ~170 | 150-200 | 轻度 (主力) |
| Aug Medium | 5,000 | ~120 | 100-150 | 中度 |
| Aug Heavy | 2,000 | ~80 | 50-100 | 重度 |
| **总计** | **20,000** | - | 50-255 | - |

### 各增强级别参数

#### Clean (无增强)
- 保持原始白底
- 覆盖真实数据中的扫描件

#### Aug Light (轻度) - 最重要
- `noise`: 20 (原来 40)
- `rotation`: 5° (原来 10°)
- `wrinkles`: 无
- `random_bw`: 0.1 (低概率黑白)
- `temperature`: 8000 (偏暖色)

#### Aug Medium (中度)
- `noise`: 40
- `rotation`: 10°
- `wrinkles`: 有 (3x3)
- `random_bw`: 0.3
- `temperature`: 20000

#### Aug Heavy (重度)
- `noise`: 60
- `rotation`: 15°
- `wrinkles`: 有 (5x5)
- `random_bw`: 0.5
- `temperature`: 40000

## 数据增强参数详解

### 当前脚本参数问题

```bash
# 02_gen_12x1_aug.sh 的问题:
--random_bw 0.2        # 20%概率转黑白,会显著降低mean
--wrinkles             # 折痕会添加阴影,降低mean
-noise 40              # 噪声会使图像变暗
-t 40000               # 高色温会使图像偏蓝/偏暗
```

### 建议的参数调整

为了让增强数据覆盖真实数据分布 (mean 150-200):

1. **降低 random_bw 概率**: 0.2 → 0.1
2. **减少折痕强度**: 或在轻度增强中完全去除
3. **降低噪声**: 40 → 20-30
4. **调整色温**: 40000 → 8000-15000 (偏暖色保持亮度)

## 后续优化方向

1. **在线增强**: 在 nnUNet 训练时添加亮度/对比度随机增强
2. **混合策略**: 将 Clean 和 Aug 按比例混合
3. **域适应**: 考虑在推理时对输入图像做预处理
